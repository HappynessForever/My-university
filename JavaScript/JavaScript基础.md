#  概述与基础语法

### JavaScript概述

JavaScript是一种运行于JavaScript解释器/引擎中的解释型脚本语言

- 运行环境
  - 独立安装的JS解释器
  - 嵌套在【浏览器】内核中JS解释器
- 解释型
  - 运行之前是不需要编译的，运行之前不会检查错误，指导碰到错误为止
- 编译型
  - 对源码进行编译，还能检查语法错误
- JS组成
  - 核心（ECMAScript）
  - 文档对象模型（DOM，Document Object Model），让JS有能力与网页进行对话
  - 浏览器对象模型（BOM，Browser Object Model），让JS有能力与浏览器进行对话
- JS特点
  - 开发简单，记事本即可
  - 无需编译，直接由JS引擎负责执行
  - 弱类型语言由数据来决定数据类型
  - 面向对象
- JavaScript基础语法
  - 浏览器内核的作用
    - 负责页面内容的渲染
    - 组成
      - 内容排版引擎解析（HTML和CSS）
      - 脚本解释引擎解析（JavaScript）
  - JS运行环境
    - 直接在浏览器Console中输入并执行JS
    - 将JS脚本嵌入在HTML页面中执行
  - 语法规范
    - 语句
      - 允许被JS引擎所解释的代码，使用分号来表示结束
      - 大小写敏感
      - 英文标点符号
      - 由表达式，关键字，运算符组成
    - 注释
      - //——单行注释
      - /* */——多行注释
